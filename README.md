## Project setup

```
npm install
```

### Run your unit tests

```
npm run test:unit
```

### Lints and fixes files

```
npm run lint
```

### E2E tests

```
npx playwright test --ui
```

Run all tests in headless mode meaning no browser will open up when running the tests. Results of the tests and test logs will be shown in the terminal.

```
npx playwright test
```


### Run local server

```
http-server -p 8080
```

### typescript compile

```
npx tsc
```
